# E-Commerce Order Management System (OMS) – BA Project

Business Analyst project showcasing BRD & FRD for an E-Commerce Order Management System.

---

## 📌 Project Overview

This project demonstrates end-to-end Business Analyst documentation for an E-Commerce Order Management System (OMS). It includes structured Business Requirement Document (BRD) and Functional Requirement Document (FRD) based on a real-world scenario.

---

## 🎯 Objective

* Understand the order lifecycle in e-commerce
* Create BRD and FRD documents
* Demonstrate Business Analyst skills

  
## 📊 Business Value

This system helps streamline order processing, reduce manual errors, and improve customer experience by enabling efficient order tracking, payment handling, and order fulfillment.

---

## 🧾 Sample User Stories

- As a customer, I want to place an order so that I can purchase products online  
- As a customer, I want to track my order so that I can check delivery status  
- As an admin, I want to manage orders so that I can ensure smooth processing
